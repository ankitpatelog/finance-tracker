# 💰 CoinTrak — Smart Expense Tracker with Graphs

**CoinTrak** is a modern, lightweight expense tracking app that helps you understand your finances visually.  
Add your daily expenses, categorize them, and see your spending habits come to life through interactive graphs and charts.

---

## 🚀 Features

- 📊 **Visual Expense Insights** — Interactive graphs to understand where your money goes.  
- 💸 **Add & Manage Expenses Easily** — Add transactions in just a few clicks.  
- 🧾 **Categorized Spending** — Track expenses by categories like Food, Travel, Utilities, etc.  
- 🗓️ **Monthly & Weekly Views** — Get detailed insights over time.  
- ☁️ **Persistent Data** — Save and load data locally (or via cloud sync, if integrated).  
- 🌙 **Dark & Light Mode** — Designed for all-day usability.  

---

## 🛠️ Tech Stack

- **Frontend:** React / Next.js  
- **Backend:** Node.js + nextauth.js 
- **Database:** MongoDB / IndexedDB  
- **Charts:**  Chart.js
